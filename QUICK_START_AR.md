# 🚀 دليل البدء السريع

## التثبيت في 5 خطوات

### 1️⃣ تحضير البيئة
```bash
# تأكد من تثبيت Node.js
node --version    # يجب أن يكون 16 أو أحدث
npm --version     # يجب أن يكون 8 أو أحدث
```

### 2️⃣ تحضير المشروع
```bash
# انسخ الملفات إلى مجلد جديد
cd special-center

# ثبّت المكتبات
npm install

# قد تستغرق هذه الخطوة 5-10 دقائق في المرة الأولى
```

### 3️⃣ التشغيل في بيئة التطوير
```bash
npm run dev

# ستظهر رسالة مثل:
# VITE v8.0.7  ready in 500ms
# 
# ➜  Local:   http://localhost:5173/
# ➜  Press h to show help
```

### 4️⃣ الوصول إلى التطبيق
- افتح المتصفح وادخل: `http://localhost:5173`
- استخدم بيانات الدخول الافتراضية:
  - **اسم المستخدم:** admin
  - **كلمة المرور:** admin123

### 5️⃣ البناء للإنتاج
```bash
# عند الانتهاء من التطوير
npm run build

# المجلد المُنتج: dist/
# يمكن نشره على Vercel أو أي سيرفر آخر
```

## 🛠️ حل المشاكل الشائعة

### ❌ خطأ: Command not found npm
**الحل:** ثبّت [Node.js](https://nodejs.org) على جهازك

### ❌ خطأ: Port 5173 is already in use
**الحل:** استخدم port مختلف
```bash
npm run dev -- --port 3000
```

### ❌ بيانات الدخول لا تعمل
**الحل:** تحقق من:
1. كتابة اسم المستخدم **admin** بدقة
2. كتابة كلمة المرور **admin123** بدقة
3. مسح cache المتصفح (Ctrl+Shift+Delete)

### ❌ البيانات تختفي عند التحديث
**الحل:** البيانات تُحفظ في localStorage
- اختر: **⚙️ الإعدادات → 💾 النسخ الاحتياطي → تصدير**
- احفظ ملف JSON بأمان
