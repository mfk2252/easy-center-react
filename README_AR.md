## 📋 نظرة عامة

**نظام إدارة المركز المتكامل** هو تطبيق ويب حديث مصمم خصيصاً لإدارة مراكز التعليم الخاص والتربية الخاصة. يجمع بين الواجهة الأنيقة والقوة الوظيفية لتوفير حل شامل لإدارة:

- 👦 بيانات الطلاب والسجلات الأكاديمية
- 👥 إدارة الموظفين والرواتب
- 📊 التقارير والإحصائيات المتقدمة
- 💳 الإدارة المالية والحسابات
- 📅 التقويم والمواعيد والجلسات
- 🤝 الشراكات والعلاقات الخارجية
- 📄 إدارة الوثائق والملفات
- ⚠️ نظام الإنذارات والنبهات الذكية

## ✨ المميزات الرئيسية

### 🔐 نظام الصلاحيات الذكي
- صلاحيات تفصيلية لكل مستخدم
- 11 صلاحية قابلة للتحديد (الرئيسية، الطلاب، الموظفون، المالية، إلخ)
- حماية كلمة سر لقسم المالية
- إدارة شاملة للأدوار (مدير، نائب، أخصائي، إداري، إلخ)

### 📊 التقارير والإحصائيات
- تقارير شاملة عن الطلاب والموظفين
- إحصائيات الحضور بفلاتر زمنية متقدمة
- تقارير مالية سنوية/نصف سنوية/شهرية
- إمكانية الطباعة من أي تقرير

### 💰 الإدارة المالية
- تتبع الإيرادات والمصروفات
- تقارير ختامية احترافية
- حسابات محاسبية متقدمة
- إدارة الرواتب والحوافز

### 🎯 البرامج والفعاليات
- تسجيل الفعاليات والأنشطة
- إرفاق صور ومستندات
- عرض تفاصيل كاملة لكل فعالية
- طباعة احترافية بشعار المركز

### 📱 واجهة متجاوبة
- تصميم جميل وحديث
- استجابة كاملة على الموبايل والتابلت
- نوافذ منبثقة محسّنة
- تجربة مستخدم سلسة

### 🖨️ نظام الطباعة المتقدم
- طباعة احترافية من أي بند
- شعار المركز متضمن في كل طبعة
- تنسيق احترافي ومتناسق
- دعم الطباعة من جميع الأقسام

### ⚠️ نظام الإنذارات الذكي
- إنذارات رسمية للموظفين
- إرسال عبر واتساب مباشر
- طباعة رسمية للإنذارات
- تتبع شامل للإنذارات

## 🛠️ التثبيت والتشغيل

### المتطلبات
- Node.js 16 أو أحدث
- npm أو yarn
- متصفح حديث (Chrome, Firefox, Safari, Edge)

### خطوات التثبيت

```bash
# 1. استنساخ المشروع أو فك ضغط الملف
cd special-center

# 2. تثبيت المكتبات
npm install

# 3. تشغيل المشروع في بيئة التطوير
npm run dev

# 4. بناء المشروع للإنتاج
npm run build

# 5. معاينة النسخة النهائية
npm run preview
```

## 📖 دليل الاستخدام

### تسجيل الدخول
1. افتح التطبيق في المتصفح
2. أدخل اسم المستخدم وكلمة المرور
3. اختر خيار "تذكر البيانات" حسب الحاجة
4. اضغط على زر "دخول"

### إدارة المستخدمين (المدير فقط)
1. اذهب إلى **الإعدادات → المستخدمون**
2. اضغط على **➕ مستخدم جديد**
3. أدخل البيانات الأساسية
4. حدد الصلاحيات المطلوبة من القائمة
5. اضغط على **💾 حفظ**

### إضافة طالب جديد
1. اذهب إلى **👦 الطلاب**
2. اضغط على **➕ طالب جديد**
3. ملء جميع البيانات المطلوبة
4. أضف صورة الطالب (اختياري)
5. اضغط على **💾 حفظ**

### تسجيل جلسة علاجية
1. اذهب إلى **👦 الطلاب**
2. اضغط على **🩺 تسجيل جلسة**
3. اختر الطالب والنوع والتاريخ
4. أضف الملاحظات والمرفقات
5. اضغط على **💾 حفظ الجلسة**

### إضافة معاملة مالية
1. اذهب إلى **🏛️ إدارة المركز → 💳 المالية**
2. أدخل كلمة السر (إن وجدت)
3. اضغط على **➕ إيراد** أو **➕ مصروف**
4. ملء البيانات
5. اضغط على **💾 حفظ**

### طباعة بند
1. انقر على البند المراد طباعته
2. اضغط على زر **🖨️ طباعة** في النافذة المنبثقة
3. ستظهر نافذة الطباعة بصيغة احترافية
4. اختر الطابعة واضغط **طباعة**

## 🔑 نسيت كلمة المرور؟

### إذا كنت المدير
1. اذهب إلى **⚙️ الإعدادات → 💾 النسخ الاحتياطي**
2. اذهب إلى قسم **نسيت كلمة المرور؟**
3. ستجد قائمة بجميع المستخدمين
4. اضغط على **🔑 تغيير** بجانب المستخدم
5. أدخل كلمة المرور الجديدة

### إذا كنت موظفاً
1. على شاشة الدخول اضغط على **🔑 نسيت كلمة المرور؟**
2. تواصل مع **المدير** لإعادة تعيين كلمة المرور
3. أو تواصل عبر البريد: `mfekry225@outlook.com`

## 💾 النسخ الاحتياطية

### تصدير البيانات
1. اذهب إلى **⚙️ الإعدادات → 💾 النسخ الاحتياطي**
2. اضغط على **📥 تصدير البيانات (JSON)**
3. سيتم تحميل ملف JSON بجميع البيانات
4. احفظه في مكان آمن

### استيراد البيانات
1. اذهب إلى **⚙️ الإعدادات → 💾 النسخ الاحتياطي**
2. اضغط على **📤 استيراج بيانات**
3. اختر ملف JSON محفوظ مسبقاً
4. تأكد من النسخة الاحتياطية قبل الاستيراج

### استيراج من Excel
1. اذهب إلى **⚙️ الإعدادات → 💾 النسخ الاحتياطي**
2. اختر **📊 استيراج من Excel (CSV)**
3. اتبع التعليمات المعروضة
4. حضر ملف CSV بالصيغة الصحيحة

## 🎨 التخصيص

### تغيير ألوان المركز
1. اذهب إلى **⚙️ الإعدادات → 🏥 المركز**
2. اختر اللون المفضل من القائمة
3. سيتم التطبيق فوراً

### إضافة شعار المركز
1. اذهب إلى **⚙️ الإعدادات → 🏥 المركز**
2. اضغط على **📷 تحميل الشعار**
3. اختر صورة الشعار
4. ستظهر في كل وثائق الطباعة

## 📊 البيانات المدعومة

### الطلاب
- البيانات الأساسية (اسم، تاريخ ميلاد، جنس)
- التشخيص والحالة الطبية
- بيانات ولي الأمر
- الصور والمرفقات
- السجل الأكاديمي

### الموظفون
- البيانات الأساسية (اسم، هاتف، بريد)
- الدور الوظيفي والصلاحيات
- الرتبة والراتب
- تاريخ التوظيف

### الجلسات العلاجية
- نوع الجلسة (تخاطب، سلوك، فيزيائي، إلخ)
- الأخصائي المسؤول
- الملاحظات والأهداف
- المرفقات والتقييمات

### المالية
- الإيرادات (رسوم، تبرعات، منح)
- المصروفات (رواتب، إيجار، فواتير)
- الرواتب والحوافز
- التقارير الختامية
