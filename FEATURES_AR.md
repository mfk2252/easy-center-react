# 📋 دليل الميزات الجديدة

## الميزات المضافة في هذا الإصدار

---

## 1️⃣ نظام الصلاحيات الذكي ⭐⭐⭐

### المشكلة القديمة
- كل مستخدم يرى جميع أقسام التطبيق
- لا توجد طريقة للتحكم في ما يراه كل موظف
- صعوبة إدارة أدوار مختلفة

### الحل الجديد
✅ **صلاحيات تفصيلية لكل مستخدم**
- 11 صلاحية قابلة للاختيار
- كل موظف يرى فقط ما له صلاحية لرؤيته
- المدير يتحكم كاملاً في الصلاحيات

### كيفية الاستخدام
```
⚙️ الإعدادات → المستخدمون → ➕ مستخدم جديد

البيانات الأساسية:
- الاسم الكامل
- اسم المستخدم  
- كلمة المرور
- الدور الأساسي

🔐 الصلاحيات التفصيلية:
☐ 📊 الرئيسية (لوحة التحكم)
☐ 👦 الطلاب
☐ 👥 الموظفون
☐ 💳 المالية
☐ 📊 التقارير
☐ ⚙️ الإعدادات
☐ 📄 الوثائق
☐ 👨‍👩‍👧 أولياء الأمور
☐ 🤝 الشراكات
☐ 🏛️ الزيارات
☐ 📅 التقويم

اختر الصلاحيات ثم اضغط 💾 حفظ
```

## 2️⃣ حماية قسم المالية 🔐

### المشكلة القديمة
- أي موظف يستطيع رؤية البيانات المالية الحساسة
- لا توجد طريقة لحماية المعلومات المالية

### الحل الجديد
✅ **كلمة سر خاصة لقسم المالية**
- عند الدخول إلى المالية يطلب كلمة سر
- تحقق ذاتي الإدارة
- يمكن تغييرها من الإعدادات

### كيفية الاستخدام
```
🏛️ إدارة المركز → 💳 المالية

🔐 حماية قسم المالية
أدخل كلمة السر:
[________]

خيارات:
✅ تأكيد
🔑 تعيين كلمة سر (إذا كانت فارغة)

تعيين كلمة سر جديدة:
- المدير يختار الكلمة الأولى
- تُحفظ محلياً وآمنة
- يمكن تغييرها لاحقاً
```

---

## 3️⃣ التقرير الختامي الذكي 📊

### المشكلة القديمة
- لا يوجد تقارير محاسبية احترافية
- الحسابات اليدوية معرضة للخطأ
- صعوبة معرفة الصافي السنوي

### الحل الجديد
✅ **تقارير ختامية بصيغة محاسبية احترافية**
- تقارير شهرية، ربع سنوية، سنوية
- حسابات تلقائية صحيحة 100%
- شعار المركز في الطباعة

### كيفية الاستخدام
```
🏛️ إدارة المركز → 💳 المالية

🎛️ اختيارات:
[شهري ▼] 📊 تقرير ختامي

النتيجة:
✅ تقرير كامل بـ:
- إجمالي الإيرادات
- إجمالي المصروفات  
- الصافي النهائي
- جدول تفصيلي لكل بند
- شعار وبيانات المركز
- تاريخ وتوقيع
```

---

## 4️⃣ نظام الإنذارات المتقدم ⚠️

### المشكلة القديمة
- لا توجد طريقة لتوثيق الإنذارات رسمياً
- صعوبة إرسال إنذارات للموظفين
- لا يمكن طباعة إنذارات رسمية

### الحل الجديد
✅ **نظام إنذارات شامل وفعال**
- إضافة إنذارات مع سبب مفصل
- إرسال عبر واتساب مباشر
- طباعة رسمية بتوقيع رسمي

### كيفية الاستخدام
```
📄 الوثائق → ⚠️ إنذارات ← ➕ إنذار جديد

البيانات:
- الموجه إليه (من قائمة الموظفين)
- نوع الإنذار (إنذار/إيقاف/إنهاء)
- التاريخ
- السبب التفصيلي
- ملاحظات إضافية

الإجراءات:
💬 إرسال واتساب ← يفتح محادثة واتس
🖨️ طباعة ← طباعة رسمية بقوة قانونية
🗑️ حذف ← حذف الإنذار
```

---

## 5️⃣ نظام الطباعة الذكي 🖨️

### المشكلة القديمة
- لا يمكن طباعة بيانات البنود الفردية
- الطباعة غير احترافية
- شعار المركز غير متضمن

### الحل الجديد
✅ **طباعة احترافية من أي بند**
- شعار المركز مضمن تلقائياً
- تنسيق احترافي ومتناسق
- يدعم جميع الأقسام

### أماكن الطباعة
```
✅ الشراكات 🤝
   - البطاقة الأساسية
   - نافذة التفاصيل الكاملة

✅ الفعاليات 🎯
   - صورة الفعالية
   - المشاركون والمسؤولون
   - الملاحظات والمرفقات

✅ المالية 💳
   - بيانات الإيراد/المصروف
   - الفئة والمبلغ
   - التفاصيل الكاملة

✅ الإنذارات ⚠️
   - صيغة رسمية قانونية
   - مساحة توقيع
   - بيانات الموظف

✅ التقارير 📊
   - تقرير كامل بتنسيق احترافي
   - جداول منظمة
   - شعار وبيانات المركز
```

---

## 6️⃣ عرض التفاصيل الموسّع 👁️

### المشكلة القديمة
- رؤية محدودة لبيانات البنود
- صعوبة الوصول لكل التفاصيل
- لا توجد نافذة معلومات كاملة

### الحل الجديد
✅ **نافذة تفاصيل كاملة لكل بند**
- اضغط على أي بند لرؤية كل التفاصيل
- تعديل سهل من النافذة
- حذف آمن مع تأكيد

### كيفية الاستخدام
```
مثال: الشراكات 🤝

الخطوة 1: انقر على الشريك
الخطوة 2: نافذة منبثقة بـ:
  - اسم الشراكة
  - النوع والقطاع
  - جهة الاتصال
  - الهاتف والإيميل
  - تاريخ البدء
  - الملاحظات الكاملة

الخطوة 3: الخيارات:
  🖨️ طباعة ← طباعة احترافية
  ✏️ تعديل ← تعديل البيانات
  🗑️ حذف ← حذف البند
  ✕ إغلاق ← إغلاق النافذة
```

## 7️⃣ استجابة محسّنة للموبايل 📱

### التحسينات
✅ نوافذ منبثقة تظهر من الأسفل
✅ أزرار أكبر وأسهل للنقر
✅ نصوص أكبر وأوضح
✅ تخطيط واحد العمود تلقائياً
✅ حفظ الشاشة عند الدخول لنافذة

### المثال: نافذة الزيارة
```
الهاتف (قديماً):
❌ نافذة صغيرة بوسط الشاشة
❌ أزرار صغيرة ومتقاربة
❌ نص ضيق وصعب القراءة
❌ اختفاء الأزرار أحياناً

الآن:
✅ نافذة تملأ الشاشة من الأسفل
✅ أزرار كبيرة وواضحة
✅ نص كبير وسهل
✅ أزرار الحفظ والإلغاء دائماً ظاهرة
```

## 8️⃣ استيراج من Excel 📊

### المشكلة القديمة
- إدخال يدوي للبيانات يستغرق وقتاً طويلاً
- احتمالية حدوث أخطاء في الكتابة
- صعوبة نقل البيانات من نظام قديم

### الحل الجديد
✅ **استيراج سريع من ملفات Excel**
- تحويل Excel إلى CSV
- تحميل مباشر من الإعدادات
- تحقق تلقائي من الصيغة

### كيفية الاستخدام
```
⚙️ الإعدادات → 💾 النسخ الاحتياطي

📊 استيراج من Excel (CSV)

خطوات التحضير:
1. افتح Excel جديد
2. السطر الأول: أسماء الأعمدة (بالإنجليزية)
   مثال: name, dob, diagnosis, parentName, parentPhone
3. من السطر الثاني: البيانات
4. احفظ كـ CSV UTF-8
5. ارفع الملف من الإعدادات

مثال لطلاب:
name,dob,diagnosis,parentName,parentPhone
أحمد محمد,2019-05-10,توحد,محمد علي,0501234567
نور أحمد,2020-03-15,تأخر لغوي,سارة علي,0502345678
```

## 9️⃣ إدارة الصلاحيات المتقدمة 🔐

### الصلاحيات المتاحة
```
1. 📊 الرئيسية (لوحة التحكم)
2. 👦 الطلاب (عرض وإضافة وتعديل)
3. 👥 الموظفون (جميع العمليات)
4. 💳 المالية (الإيرادات والمصروفات)
5. 📊 التقارير (عرض وطباعة)
6. ⚙️ الإعدادات (تعديل إعدادات المركز)
7. 📄 الوثائق (الوثائق والإنذارات)
8. 👨‍👩‍👧 أولياء الأمور (سجلات التواصل)
9. 🤝 الشراكات (إضافة وتعديل)
10. 🏛️ الزيارات (الزيارات الخارجية)
11. 📅 التقويم (الجلسات والمواعيد)
```

### مثال: موظف متخصص
```
الموظف: علي - أخصائي تخاطب

الصلاحيات المقترحة:
✅ الرئيسية
✅ الطلاب
❌ الموظفون
❌ المالية
✅ التقارير
❌ الإعدادات
❌ الوثائق
❌ أولياء الأمور
❌ الشراكات
❌ الزيارات
✅ التقويم

النتيجة: علي يرى فقط بيانات طلابه وجلساته
```
